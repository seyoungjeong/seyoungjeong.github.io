# seyoungjeong.github.io

This is the repository for my personal blog.

## Content Creation

I use **Obsidian** to create and edit posts. 

To ensure the best writing experience and use the configured settings:
1. Open this repository's folder directly in Obsidian.
2. The folder contains settings configured for a **minimal focus mode**.
3. You can toggle this focus mode on and off by pressing `Shift` + `Ctrl` + `F` in Obsidian.
